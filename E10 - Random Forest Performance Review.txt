Por lo general al realizar algoritmos de clasificación, las personas tienden a realizar los que mejor entienden o se adecuan a sus conocimientos dependiendo del área de la cual procedan, como lo son Matemáticas, Estadística o Computación. En cada caso las diferentes ramas han generado sus algoritmos que permiten realizar esta tarea de una forma un poco más eficiente, esto ha generado que al ser el Machine learning una rama que combina todas estas disciplinas, el científico de datos cuente con una amplia gama de algoritmos para lograr este fin lo que nos lleva a la pregunta ¿Cuál es mejor?
Para poder responder esta pregunta, un grupo de científicos se encargó de analizar cada uno de los algoritmos con los que contamos actualmente para realizar tareas de clasificación. Este estudio demostró que, al analizarse una información determinada con cada uno de estos algoritmos, estos demostraban tener una mejor efectividad dependiendo del tipo de información que se le ingresara los únicos algoritmos que demostraban tener un rendimiento adecuado para todos los datos eran el Random Forest y El Suported Vector Machine con Kernel Gaussiano.
Esto nos indica dos cosas, el científico de datos debe estar en la capacidad de identificar qué tipo de algoritmo debe usar cuando se desea analizar un grupo de datos, aunque siempre una buena opción, es apoyarse en los dos algoritmos mejor raqueados a fin de poder hacer una comparación adecuada de los resultados con el fin de generar un rango base estable para el desarrollo de los estudios
